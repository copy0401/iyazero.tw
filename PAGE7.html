傷害/命中率/生命補滿公式

物理傷害公式 = 武器傷害值 x 技能加成 x 物理抗性 x 致命攻擊 + 元素傷害

武器傷害值 = 基本傷害值 x (1+增強傷害) + (增加最大傷害) + (增加最小傷害)

物理抗性 = (1-敵人的物理抗性)

致命攻擊 = (1 + 技能加成致命攻擊 + 裝備加成致命攻擊)

　

元素傷害算法

若受到元素傷害100
抗性95%實際傷害 100 x (100%-95%)=5
抗性75%實際傷害 100 x (100%-75%)=25
抗性0%實際傷害 100 x (100%-0%)=100
抗性-70%實際傷害 100 x [100%-(-70%)]=170

　

命中率 = 攻擊者準確率/(攻擊者準確率+防禦者防禦力) x 攻擊者等級/(攻擊者等級+防禦者等級) x 2 x 100

備註: 防禦越高只會降低敵人命中率 並不會減少傷害

　

生命每秒補滿公式 = (25 x 生命補滿/256)
