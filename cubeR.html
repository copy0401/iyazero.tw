符文方塊公式

3個同樣的符石 = 1個高一等級的符石
3 El 艾爾 (1) = Eld 艾德 (2)
3 Eld 艾德 (2) = Tir 特爾 (3)
3 Tir 特爾 (3) = Nef 那夫 (4)
3 Nef 那夫 (4) = Eth 愛斯 (5)
3 Eth 愛斯 (5) = Ith 伊司 (6)
3 Ith 伊司 (6) = Tal 塔爾 (7)
3 Tal 塔爾 (7) = Ral 拉爾 (8)
3 Ral 拉爾 (8) = Ort 歐特 (9)
3 Ort 歐特 (9) = Thul 書爾 (10)

3 Thul 書爾 ( 10 ) + 1 碎裂的黃寶石 = Amn 安姆 ( 11 )

3 Amn 安姆 ( 11 ) + 1 碎裂的紫寶石 = Sol 索爾 ( 12 )

3 Sol 索爾 ( 12 ) + 1 碎裂的藍寶石 = Shael 夏 ( 13 )

3 Shael 夏 ( 13 ) + 1 碎裂的紅寶石 = Dol 多爾 ( 14 )
　

以下符文升級公式僅限於天梯與單機版

3 Dol 多爾 ( 14 ) + 1 碎裂的綠寶石 = Hel 海爾 ( 15 )

3 Hel 海爾 ( 15 ) + 1 碎裂的鑽石 = Io 破 ( 16 )

3 Io 破 ( 16 ) + 1 裂開的黃寶石 = Lum 盧姆 ( 17 )

3 Lum 盧姆 ( 17 ) + 1 裂開的紫寶石 = Ko 科 ( 18 )

3 Ko 科 ( 18 ) + 1 裂開的藍寶石 = Fal 法爾 ( 19 )

3 Fal 法爾 ( 19 ) + 1 裂開的紅寶石 = Lem 藍姆 ( 20 )

3 Lem 藍姆 ( 20 ) + 1 裂開的綠寶石 = Pul 普爾 ( 21 )

2 Pul 普爾 ( 21 ) + 1 裂開的鑽石 = Um 烏姆 ( 22 )

2 Um 烏姆 ( 22 ) + 1 黃寶石 = Mal 馬爾 ( 23 )

2 Mal 馬爾 ( 23 ) + 1 紫寶石 = Ist 伊司特 ( 24 )

2 Ist 伊司特 ( 24 ) + 1 藍寶石 = Gul 古爾 ( 25 )

2 Gul 古爾 ( 25 ) + 1 紅寶石 = Vex 伐克斯 ( 26 )

2 Vex 伐克斯 ( 26 ) + 1 綠寶石 = Ohm 歐姆 ( 27 )

2 Ohm 歐姆 ( 27 ) + 1 鑽石 = Lo 羅 ( 28 )

2 Lo 羅 ( 28 ) + 1 無瑕疵的黃寶石 = Sur 瑟 ( 29 )

2 Sur 瑟 ( 29 ) + 1 無瑕疵的紫寶石 = Ber 貝 ( 30 )

2 Ber 貝 ( 30 ) + 1 無瑕疵的藍寶石 = Jah 喬 ( 31 )

2 Jah 喬 ( 31 ) + 1 無瑕疵的紅寶石 = Cham 查姆 ( 32 )

2 Cham 查姆 ( 32 ) + 1 無瑕疵的綠寶石 = Zod 薩德 ( 33 )
