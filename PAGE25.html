1.如何修改傳送WP

用Master提取patch_d2.mpq

然後選擇data/global/excel

對右方的Levels.txt按滑鼠右鍵

選擇Extract將他提取出來

然後 用Excel開啟它(讀取txt檔)

在Waypoint(倒數第21個選項)

255 = 無
0 = Act1Wp1
1 = Act1Wp2... 以此類推

Waypoint旁邊有地圖的名稱
也可在LevelName與LevelWarp內修改地圖文字與進入地圖的文字
不過並不影響地圖本身

例如要將冰冷之原取消改成鮮血荒地
請將4號Act 1 - Wilderness 2(Cold Plains)中的Waypoint
把1改成255
然後將3號Act 1 - Wilderness 1(Blood Moor)中的Waypoint
把255改1
儲存(save txt)覆蓋即可

你可以下載我的模組後重置

重置方法:只需將C:\Program Files\Diablo II\data\global\excel內中的資料全部刪除

即變回最初始的暗黑 然後放入你修改的檔案(Levels.txt)即可

然後 用Excel開啟它(讀取txt檔)

以模式為例 如果要修改牛關的數量

先看到編號40Act 1 - Moo Moo Farm

找到MonDen.MonDen[N].MonDen[H](大約在中間的位子)

這3欄分別代表普通.惡夢.地獄牛關圖的怪物密集度

預設值為800 模式是修改為2200(正規遊戲中出現的最大值)

當然 你可以改更高 不過我沒嘗試過就是了

你可以下載我的模組後重置

重置方法:只需將C:\Program Files\Diablo II\data\global\excel內中的資料全部刪除

即變回最初始的暗黑 然後放入你修改的檔案(cubemain.txt)即可

　

8.如何修改怪物掉下寶物類型

用Master提取patch_d2.mpq

然後選擇data/global/excel

對右方的monstats.txt按滑鼠右鍵

選擇Extract將他提取出來

然後 用Excel開啟它(讀取txt檔)

以模式為例 如果要改變地獄之牛的掉下類型

請找到編號392hellbovine

然後找到TreasureClass1-4.TreasureClass1-4[N].TreasureClass1-4[H](尾端位子)

這12個選項分別代表普通地獄之牛掉落類型1-4.惡夢地獄之牛掉落類型1-4.地獄地獄之牛掉落類型1-4

最多可以選擇4種類型

你可以依照自己的喜好填入(若改成DC模式但怪物等級不滿110的前題 則會掉出普通魔法護身符)

如果 要改成每隻牛都是地獄巴爾掉寶只要將裡面都填入Baalq (H)

儲存(save txt)覆蓋即可

你可以下載我的模組後重置

重置方法:只需將C:\Program Files\Diablo II\data\global\excel內中的資料全部刪除

即變回最初始的暗黑 然後放入你修改的檔案(cubemain.txt)即可

　

9.如何新增變身類型

用Master提取patch_d2.mpq

然後選擇data/global/excel

對右方的states.txt按滑鼠右鍵

選擇Extract將他提取出來

然後 用Excel開啟它(讀取txt檔)

用Add Row新增一欄全新的

state -> new01 (名稱)

qroup -> 3 (組隊影響)

transform -> 1 (是否是變形狀態)

disquise -> 1 (是否改變圖像)

castoverlay -> teleport (開始動畫 預設為傳送)

removerlay -> teleport (結束動畫 預設為傳送)

qfxtype -> 1 (變身圖形類型 預設為怪物)

qfxclass -> 256 (變身id 預設為依族爾)

儲存(save txt)

然後覆蓋上去

你可以下載我的模組後重置

重置方法:只需將C:\Program Files\Diablo II\data\global\excel內中的資料全部刪除

即變回最初始的暗黑 然後放入你修改的檔案(states.txt)即可

完成後 此屬性為stat的編號185變身

用修改器加入即可
